# ROAST & SOUL — Coffee Shop Website

## Проект
Сайт кофейни для портфолио. Чистый HTML/CSS/JS без фреймворков.

## Структура
```
roast-and-soul/
├── index.html   # Вся разметка
├── style.css    # Все стили
├── main.js      # Вся логика
└── CLAUDE.md
```

## Запуск
Сервер запускается через `.claude/launch.json` (PowerShell HTTP-сервер на порту 3457).
Открыть: `http://localhost:3457`

## Дизайн
- **Стиль:** тёмный/уютный (dark cozy)
- **Палитра:** `#0C0804` (фон) · `#C8934A` (золото) · `#F0E6D0` (кремовый)
- **Шрифты:** Playfair Display (заголовки) + Inter (текст) — Google Fonts

## Разделы (в порядке scroll-snap)
1. **Hero** — полноэкранный, фото, вращающаяся печать "5+ лет"
2. **О нас** — две overlapping фотографии + статистика
3. **Меню** — 4 таба (Эспрессо / Фильтр / Колд брю / Еда)
4. **Галерея** — 5 фото, клик открывает модальное окно с листанием
5. **Ценности** — "Три принципа которым мы верны" на фоне кофе
6. **Контакты** — информация + форма обратной связи
7. **Футер** — логотип, навигация, соцсети

## Ключевые фичи
- **Scroll-snap** (`scroll-snap-type: y mandatory`) — каждая секция на 100vh
- **Модалка галереи** — стрелки ←→, Esc, клик на фон закрывает
- **Бургер-меню** — на мобильных (< 640px)
- **Вращающаяся SVG-печать** — анимация 18s linear infinite
- **Fade-in анимации** — IntersectionObserver при скролле
